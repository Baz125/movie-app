Movie API

The Movie Api is built on the MERN stack. It allows the user to access a database of Movies, returns data on things like genres actors directors etc. Users can also create a profile to save their own favourites.

Business Logic URL HTTP Method Request body data format Response body data format
Return a list of ALL movies /movies GET None Successful GET request returning data on all the movies
Return data (description, genre, director, image URL, whether it is featured or not) about a single movie by title /movies/:title GET None Successful GET request returning all data on a single movie
Return data about a genre (description) by name/title (e.g., “Thriller”); /movies/genres/:name GET None Successful GET request returning data on the genre of a single movie
Return data about a director (bio, birth year, death year) by name; /movies/directors/:name GET None Successful GET request returning all data on a single director by name
