Movie API

The Movie Api is built on the MERN stack. It allows the user to access a database of Movies, returns data on things like genres actors directors etc. Users can also create a profile to save their own favourites.

Business Logic URL HTTP Method Request body data format Response body data format

Return data about a genre by name (e.g., “Thriller”); /movies/genres/:name GET None A JSON object containing a genre and its attributes, example: 
{
  "Name": "Thriller",
  "Description": "Thriller film, also known as suspense film or suspense thriller, is a broad film genre that involves excitement and suspense in the audience."

Return data about a director (bio, birth year, death year) by name; /movies/directors/:name GET None A JSON object containing a director and its attributes, example: 
{
  "Name": "Jonathan Demme",

Allow new users to register /users POST A JSON object containing all required user information, example: 
{
  "Username": "celia",
  "Password": "ahern",
  "Email": "author@books.com",
  "Birthday": "87654"
  }
